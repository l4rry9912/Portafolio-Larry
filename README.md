# Portafolio Personal

Este es mi portafolio personal donde muestro algunos de los proyectos que he desarrollado durante mi formación como desarrollador.

Soy graduado en Desarrollo de Aplicaciones Multiplataforma (DAM) y actualmente estoy cursando Desarrollo de Aplicaciones Web (DAW), donde sigo ampliando mis conocimientos en desarrollo de software y tecnologías web.

Me interesa el mundo de la programación y el aprendizaje continuo. A lo largo de mi formación he trabajado con diferentes lenguajes y herramientas que me han permitido construir proyectos funcionales y mejorar mis habilidades como desarrollador.

Este portafolio está creado con el objetivo de mostrar algunos de mis proyectos, conocimientos y evolución como desarrollador.

## Tecnologías utilizadas

- React
- Tailwind
- JavaScript
- HTML
- CSS
- Canvas animations (Zenitho background)
- React Bits (animaciones)

## Características

- Diseño responsive  
- Animaciones al hacer scroll  
- Fondo animado en header y footer  
- Sección de proyectos con carrusel  
- Información académica y profesional
